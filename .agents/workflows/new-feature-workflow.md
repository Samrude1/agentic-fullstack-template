# New Feature Workflow

1. **Architect** (`/architect`):
   - Plan the feature. Check `architecture.md` to ensure compliance.
   - Create the `implementation_plan.md` artifact for approval.
2. **Develop**:
   - Write the code, strictly following the approved plan.
3. **Review** (`/review`):
   - Ensure the feature is fully built and output the review to `walkthrough.md`.
4. **Imprint** (`/imprint` / `/remember` save):
   - Update any necessary context files (e.g., `database-schema.md` or `ui-registry.md`) and save memory.
